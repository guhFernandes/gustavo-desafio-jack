Introdução

Bem-vindo à documentação da Minha Aplicação. Aqui você encontrará tudo o que precisa para instalar, configurar e utilizar o sistema.

Instalação

Para instalar a aplicação, execute o seguinte comando:

$ git clone https://github.com/usuario/minha-aplicacao.git

Configuração

Para configurar a aplicação, edite o arquivo config.json e adicione as seguintes informações: 

{
    "porta": 8080,
    "bancoDeDados": "mysql"
}

Uso

Após configurar a aplicação, execute o seguinte comando para iniciá-la:

$ npm start

API

A aplicação expõe uma API RESTful. Veja abaixo um exemplo de como fazer uma requisição:

GET /api/v1/usuarios
